# MKLab Homebrew Tap

Homebrew formulae for [MKLab](https://github.com/mklab-se) projects.

## Install

```bash
brew install mklab-se/tap/hoist
```

Or add the tap first:

```bash
brew tap mklab-se/tap
brew install hoist
```

## Available Formulae

| Formula | Description |
|---|---|
| [hoist](https://github.com/mklab-se/hoist) | Configuration-as-code CLI for Azure AI Search |
