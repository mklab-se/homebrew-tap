# MKLab Homebrew Tap

Homebrew formulae for [MKLab](https://github.com/mklab-se) projects.

## Install

```bash
brew install mklab-se/tap/<formula>
```

Or add the tap first:

```bash
brew tap mklab-se/tap
brew install <formula>
```

## Available Formulae

| Formula | Description |
|---|---|
| [cosq](https://github.com/mklab-se/cosq) | A CLI to query your Azure Cosmos DB instances |
| [hoist](https://github.com/mklab-se/hoist) | Configuration-as-code CLI for Azure AI Search |
